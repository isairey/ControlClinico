# 🏥 Sistema de Control Clínico

Sistema web desarrollado para la administración y control integral de clínicas, enfocado en la gestión de pacientes, citas médicas e historial clínico.

---

## 📌 Descripción

El **Sistema de Control Clínico** es una solución diseñada para digitalizar y optimizar los procesos internos de una clínica o consultorio médico.

Permite centralizar la información médica, mejorar la atención al paciente y facilitar la gestión administrativa.

Actualmente se encuentra en funcionamiento en **4 clínicas**, apoyando operaciones diarias de manera eficiente.

---

## 🛠 Tecnologías Utilizadas

El sistema está desarrollado con:

- **PHP** → Backend y lógica del sistema  
- **Blade** → Motor de plantillas  
- **HTML** → Estructura de interfaces  
- **CSS** → Estilos visuales  
- **JavaScript** → Interactividad  

---

## ⚙️ Funcionalidades Principales

### 👨‍⚕️ Gestión de Pacientes
- Registro de nuevos pacientes  
- Edición y actualización de datos  
- Búsqueda y consulta de información  

### 📄 Historial Clínico
- Registro de consultas médicas  
- Diagnósticos y tratamientos  
- Seguimiento del paciente  

### 📅 Gestión de Citas
- Agendamiento de citas  
- Control de horarios  
- Seguimiento de consultas  

### 🔐 Autenticación y Seguridad
- Inicio de sesión  
- Control de acceso por usuario  
- Protección de datos  

### 📊 Administración
- Organización de información clínica  
- Control de operaciones internas
